# gurudodesconto

Site WordPress do **Guru do Desconto** — reviews comparativos e promoções de afiliados (Mercado Livre, Shopee, Amazon).

## Início rápido

Todo o site está em [`public_html/`](public_html/). Consulte o guia completo de instalação na Hostinger:

**[public_html/INSTALACAO-HOSTINGER.md](public_html/INSTALACAO-HOSTINGER.md)**

## O que está incluído

- WordPress 6.x (core completo)
- Tema customizado `guru-do-desconto` com UI da marca
- Seção de reviews com links de afiliado
- CTA para grupo WhatsApp (hero + botão flutuante)
- SEO: Schema.org, Open Graph, sitemap, meta tags
- 3 reviews de exemplo para cada marketplace

## Imagens da marca

As imagens originais estão em [`images/`](images/) e foram copiadas para o tema em `public_html/wp-content/themes/guru-do-desconto/assets/images/`.
